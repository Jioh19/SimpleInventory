# SimpleInventory

SimpleInventory is a basic C# console application for managing a simple inventory of products. It allows users to add, view, and manage products through a menu-driven interface.

## Features
- Add new products to the inventory
- View all products
- Update product information
- Delete products
- Simple, menu-based navigation

## Getting Started

### Prerequisites
- [.NET 9.0 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/9.0)

### Building the Project
1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```sh
   cd SimpleInventory
   ```
3. Build the solution:
   ```sh
   dotnet build
   ```

### Running the Application
1. From the project directory, run:
   ```sh
   dotnet run --project SimpleInventory
   ```

## Project Structure
- `Inventory.cs` - Handles inventory logic and product management
- `Product.cs` - Defines the product model
- `Menu.cs` - Manages the menu and user interaction
- `Program.cs` - Entry point of the application

## Usage
Follow the on-screen menu to add, view, update, or delete products from the inventory.
